def load_data():
    animal = {"cat":1,"dog":0}
    X=[]
    y=[] 
    global FILELIST
    FILELIST=[]
    for key in FOLDER.keys():
        directory = DATADIR+FOLDER[key]
        files = os.listdir(directory)
        FILELIST += sorted(files)
        Xi=[]
        for fn in sorted(files):
            x = np.load(directory+fn)
            d = x.shape[2]-3*x.shape[1]
            d1 = int(round(d/2))
            d2 = d-d1
            xs = x.reshape((3*x.shape[1],x.shape[2]))[:,d1:x.shape[2]-d2]
            Xi.append(xs)
            yi = list(np.ones(len(Xi))*animal[key])
        X+=Xi
        y+=yi
    return np.array(X), np.array(y).T

Results:¶

  1. I created matriex & images with two cases after trimming first coefficients: (1) rescaling each mfcc/delta/delta2 matrix by its own max absolute amplitude. (2) rescaling mfcc/delta/delta2 all at once by global max of amplitude.
  2. As shown above, it seems there are more features noticible by human eyes when each matrix is rescaled individually, and the ones that are rescaled in group have hard to notice delta and delta2 patches.
  3. I ran 1 layer 2dconv but there was no difference in performance to both cases: 80% precision and 80% accuracy.
